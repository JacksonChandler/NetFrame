# netframe

Cyberpunk CSS component library — terminal green, hard edges, CRT scanlines.

```sh
npm install netframe
pnpm add netframe
yarn add netframe
```

```js
import 'netframe/dist/netframe.css';
```

Full documentation: [netframe.cybear.dev](https://netframe.cybear.dev)
