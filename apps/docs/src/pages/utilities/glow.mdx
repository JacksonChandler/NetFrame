# Glow

Hard pixel-offset shadows and ambient glow utilities. NetFrame uses solid-color offset shadows — never soft drop shadows.

## Class reference

| Class | Description |
|---|---|
| `nf-shadow-primary` | 4px offset shadow — Terminal Green |
| `nf-shadow-secondary` | 4px offset shadow — Cyber Magenta |
| `nf-shadow-tertiary` | 4px offset shadow — Electric Cyan |
| `nf-shadow-error` | 4px offset shadow — Error color |
| `nf-glow-primary` | Ambient glow — primary |
| `nf-glow-secondary` | Ambient glow — secondary |
| `nf-glow-tertiary` | Ambient glow — tertiary |
| `nf-text-glow-primary` | Text shadow — primary |
| `nf-text-glow-secondary` | Text shadow — secondary |
| `nf-text-glow-tertiary` | Text shadow — tertiary |
| `nf-border-glow-primary` | Border glow — primary |
| `nf-border-glow-secondary` | Border glow — secondary |
| `nf-border-glow-tertiary` | Border glow — tertiary |
