# Scanlines

CRT scanline overlays for authentic hardware-display texture. The `.nf-crt` class applies a full-screen fixed overlay — the "soul" of the design system.

## Full-screen CRT overlay

Apply `.nf-crt` to `<body>` (or any fixed-position container) for a global scanline effect. The docs site uses this by default.

```html
<body class="nf-crt">
  <!-- your content -->
</body>
```

The overlay uses a fixed `::before` pseudo-element with alternating green/magenta/cyan phosphor channels at 3%, 1%, and 3% opacity — matching the original CRT phosphor emission spectrum.

## Class reference

| Class | Description |
|---|---|
| `nf-crt` | Full-screen fixed CRT overlay on `::before`. Add to `<body>`. |
| `nf-scanlines` | Inline scanline overlay scoped to the element. |
| `nf-pixel-grid` | Dot-matrix pixel grid texture. |
