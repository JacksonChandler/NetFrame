## Class reference

| Class | Description |
|---|---|
| `nf-flex-center` | `display: flex; align-items: center; justify-content: center` |
| `nf-truncate` | Single-line truncation with ellipsis |
| `nf-uppercase` | `text-transform: uppercase` |
| `nf-tracking-display` | `letter-spacing: 0.05em` |
| `nf-text-primary` | Color: `--nf-color-primary` |
| `nf-text-secondary` | Color: `--nf-color-secondary` |
| `nf-text-tertiary` | Color: `--nf-color-tertiary` |
| `nf-text-muted` | Color: `--nf-color-text-muted` |
| `nf-text-subtle` | Color: `--nf-color-text-subtle` |
| `nf-font-display` | Font: Space Grotesk |
| `nf-font-body` | Font: JetBrains Mono |
| `nf-font-label` | Font: VT323 |
| `nf-visually-hidden` | Accessible hide (screen reader only) |
| `nf-border-primary` | Border: `--nf-color-primary` |
| `nf-border-secondary` | Border: `--nf-color-secondary` |
| `nf-border-tertiary` | Border: `--nf-color-tertiary` |
