# Theming

NetFrame emits every design value as a CSS custom property on `:root`. Override any token to retheme without touching a Sass file or rebuilding anything.

## Override the palette

```css
/* your-theme.css */
:root {
  --nf-color-primary:   #bd00ff;  /* swap green for purple */
  --nf-color-secondary: #ff6b00;  /* orange instead of magenta */
}
```

Scope overrides to a class to support multiple themes on one page:

```css
.theme-purple {
  --nf-color-primary: #bd00ff;
}
```
