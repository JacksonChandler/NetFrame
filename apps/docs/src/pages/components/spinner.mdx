## Class reference

| Class | Description |
|---|---|
| `nf-spinner` | Base — 1.5rem, primary top border, spinning |
| `nf-spinner-sm` | 1rem |
| `nf-spinner-lg` | 2.5rem |
| `nf-spinner-xl` | 4rem |
| `nf-spinner-secondary` | Magenta top border |
| `nf-spinner-tertiary` | Tertiary top border |
| `nf-spinner-error` | Error red top border |
| `nf-spinner-block` | Centered flex column wrapper |
| `nf-spinner-label` | Blinking text label inside `.nf-spinner-block` |
