## Class reference

| Class | Description |
|---|---|
| `nf-pagination` | `<ol>` wrapper — flex row, no gap |
| `nf-page-item` | `<li>` wrapper — collapses adjacent right borders |
| `nf-page-item-active` | Fills the page link with primary color |
| `nf-page-item-disabled` | Dims and disables pointer events |
| `nf-page-ellipsis` | Non-interactive spacer item |
| `nf-page-link` | The anchor or span inside each item |
| `nf-page-prev` | Prev label styling |
| `nf-page-next` | Next label styling |
