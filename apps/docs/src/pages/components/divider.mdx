Explicit structural separators. Use sparingly — prefer tonal surface shifts for subtle grouping.

## Class reference

| Class | Description |
|---|---|
| `nf-divider` | Horizontal rule — 2px border-top |
| `nf-divider-primary` | Primary green accent |
| `nf-divider-secondary` | Secondary magenta accent |
| `nf-divider-tertiary` | Tertiary purple accent |
| `nf-divider-vertical` | Inline vertical rule — 1em tall |
| `nf-divider-labeled` | Flex row with line on each side of a label |
| `nf-divider-label` | Label text inside `.nf-divider-labeled` |
