## Class reference

| Class | Description |
|---|---|
| `nf-progress` | Bar track — default 4px height |
| `nf-progress-bar` | Filled portion — set `width` inline |
| `nf-progress-sm` | 2px height |
| `nf-progress-lg` | 8px height |
| `nf-progress-secondary` | Magenta bar |
| `nf-progress-tertiary` | Tertiary bar |
| `nf-progress-error` | Error bar |
| `nf-progress-indeterminate` | Sliding animation — no width needed |
| `nf-progress-labeled` | Flex wrapper for bar + value text |
| `nf-progress-value` | Percentage label inside `.nf-progress-labeled` |
| `nf-progress-ring` | SVG ring container — set `--nf-ring-progress` (0–100) |
| `nf-ring-track` | SVG `<circle>` — background ring |
| `nf-ring-fill` | SVG `<circle>` — filled arc |
| `nf-ring-label` | Center text label |
