## Class reference

| Class | Description |
|---|---|
| `nf-breadcrumb` | `<ol>` wrapper — flex row with `/` separators |
| `nf-breadcrumb-item` | `<li>` — adds `/` separator via `::after` (except last) |
| `nf-breadcrumb-link` | Anchor or span inside each item |
| `nf-breadcrumb-current` | Highlights the current page — disables pointer events |
