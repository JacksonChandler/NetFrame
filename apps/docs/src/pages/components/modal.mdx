## JavaScript

```js
const overlay = document.getElementById('modal-overlay');
const modal   = overlay.querySelector('.nf-modal');
const trigger = document.getElementById('open-modal');
const close   = modal.querySelector('.nf-modal-close');

function openModal() {
  overlay.classList.remove('nf-hidden');
  overlay.removeAttribute('aria-hidden');
  modal.focus();
}

function closeModal() {
  overlay.classList.add('nf-hidden');
  overlay.setAttribute('aria-hidden', 'true');
  trigger.focus();
}

trigger.addEventListener('click', openModal);
close.addEventListener('click', closeModal);
overlay.addEventListener('click', (e) => {
  if (e.target === overlay) closeModal();
});
document.addEventListener('keydown', (e) => {
  if (e.key === 'Escape') closeModal();
});
```

## Class reference

| Class | Description |
|---|---|
| `nf-modal-overlay` | Fixed full-screen backdrop — hide with `nf-hidden` or `aria-hidden="true"` |
| `nf-modal` | Dialog panel — default max-width 32rem |
| `nf-modal-sm` | 20rem |
| `nf-modal-lg` | 48rem |
| `nf-modal-xl` | 64rem |
| `nf-modal-full` | Full viewport |
| `nf-modal-header` | Top section with title and close button |
| `nf-modal-title` | Title text |
| `nf-modal-close` | Close `×` button |
| `nf-modal-body` | Scrollable content area |
| `nf-modal-footer` | Bottom action row |
