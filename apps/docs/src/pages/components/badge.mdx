## Dot badge

Add `.nf-badge-dot` to any element to attach a notification indicator dot in the top-right corner:

## Class reference

| Class | Description |
|---|---|
| `nf-badge` | Base — required on all badges |
| `nf-badge-primary` | Terminal Green fill |
| `nf-badge-secondary` | Cyber Magenta fill |
| `nf-badge-tertiary` | Tertiary fill |
| `nf-badge-error` | Error red fill |
| `nf-badge-outline` | Transparent with primary border |
| `nf-badge-ghost` | Low-opacity primary tint, no border |
| `nf-badge-dot` | Adds a red notification dot — applies to the parent element |
