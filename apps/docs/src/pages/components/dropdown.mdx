# Dropdown

Absolutely-positioned menus anchored to a trigger. Toggle visibility with `aria-hidden` or `.nf-hidden` on the menu.

## Right-aligned

Add `.nf-dropdown-menu-right` to align the menu to the right edge of the trigger:

```html
<ul class="nf-dropdown-menu nf-dropdown-menu-right" role="menu">
  ...
</ul>
```

## JavaScript

```js
document.querySelectorAll('.nf-dropdown').forEach((dropdown) => {
  const trigger = dropdown.querySelector('.nf-dropdown-trigger');
  const menu    = dropdown.querySelector('.nf-dropdown-menu');

  trigger.addEventListener('click', (e) => {
    e.stopPropagation();
    const open = trigger.getAttribute('aria-expanded') === 'true';
    trigger.setAttribute('aria-expanded', String(!open));
    menu.toggleAttribute('aria-hidden', open ? null : true);
  });

  // Close on outside click
  document.addEventListener('click', () => {
    trigger.setAttribute('aria-expanded', 'false');
    menu.setAttribute('aria-hidden', 'true');
  });
});
```

## Class reference

| Class | Description |
|---|---|
| `nf-dropdown` | `position: relative` wrapper |
| `nf-dropdown-trigger` | Button with caret indicator |
| `nf-dropdown-menu` | Absolute menu panel — hide with `aria-hidden="true"` or `nf-hidden` |
| `nf-dropdown-menu-right` | Right-aligns the menu |
| `nf-dropdown-item` | Menu row — button or anchor |
| `nf-dropdown-item-danger` | Error color variant |
| `nf-dropdown-divider` | `<li>` separator between item groups |
| `nf-dropdown-label` | `<li>` group header — non-interactive |
