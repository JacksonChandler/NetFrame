# Card

Cards are surface containers. Compose them with `.nf-card-header`, `.nf-card-body`, and `.nf-card-footer`.

## Base

<ComponentDemo stacked title="Base card" code={`<div class="nf-card" style="max-width: 20rem;">
  <div class="nf-card-header">
    <span class="nf-card-title">NODE_STATUS</span>
  </div>
  <div class="nf-card-body">
    All systems nominal. Connection stable.
  </div>
  <div class="nf-card-footer">
    <button class="nf-button nf-button-ghost nf-button-sm">DETAILS</button>
  </div>
</div>`}>
  <div class="nf-card" style="max-width: 20rem; width: 100%;">
    <div class="nf-card-header">
      <span class="nf-card-title">NODE_STATUS</span>
    </div>
    <div class="nf-card-body">All systems nominal. Connection stable.</div>
    <div class="nf-card-footer">
      <button class="nf-button nf-button-ghost nf-button-sm">DETAILS</button>
    </div>
  </div>
</ComponentDemo>
