## As anchor

Any element works — use `role="button"` on non-button elements:

```html
<a href="/dashboard" class="nf-button nf-button-primary">
  ENTER SYSTEM
</a>
```

## Class reference

| Class | Description |
|---|---|
| `nf-button` | Base — required on all buttons |
| `nf-button-primary` | Terminal Green fill |
| `nf-button-secondary` | Cyber Magenta fill |
| `nf-button-ghost` | Transparent, green border |
| `nf-button-danger` | Error color fill |
| `nf-button-outline` | Transparent, muted border |
| `nf-button-sm` | Small size |
| `nf-button-lg` | Large size |
| `nf-button-icon` | Square icon-only button |
