## Class reference

| Class | Description |
|---|---|
| `nf-avatar` | Base container — default 2.5rem, primary border |
| `nf-avatar-initials` | Text fallback inside the avatar |
| `nf-avatar-xs` | 1.5rem |
| `nf-avatar-sm` | 2rem |
| `nf-avatar-lg` | 3.5rem |
| `nf-avatar-xl` | 5rem |
| `nf-avatar-secondary` | Secondary border + initials color |
| `nf-avatar-tertiary` | Tertiary border + initials color |
| `nf-avatar-status` | Status dot — position absolute, bottom-right |
| `nf-avatar-status-online` | Primary green dot |
| `nf-avatar-status-away` | Warning yellow dot |
| `nf-avatar-status-busy` | Error red dot |
| `nf-avatar-status-offline` | Subtle grey dot |
| `nf-avatar-group` | Flex wrapper that overlaps children |
