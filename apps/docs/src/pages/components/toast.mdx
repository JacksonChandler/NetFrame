## Usage

Append toasts into a fixed `.nf-toast-region` container, then remove them after a timeout:

```html
<div class="nf-toast-region nf-toast-top-right" id="toast-region" aria-live="polite"></div>
```

```js
function showToast({ title, message, type = 'info', duration = 4000 }) {
  const region = document.getElementById('toast-region');
  const toast = document.createElement('div');
  toast.className = `nf-toast nf-toast-${type}`;
  toast.innerHTML = `
    <div class="nf-toast-content">
      <div class="nf-toast-title">${title}</div>
      <div class="nf-toast-message">${message}</div>
    </div>
    <button class="nf-toast-close" aria-label="Dismiss">✕</button>
  `;

  const close = toast.querySelector('.nf-toast-close');
  const dismiss = () => {
    toast.classList.add('nf-toast-dismissing');
    toast.addEventListener('animationend', () => toast.remove(), { once: true });
  };

  close.addEventListener('click', dismiss);
  region.appendChild(toast);
  setTimeout(dismiss, duration);
}
```

## Region positions

| Class | Position |
|---|---|
| `nf-toast-top-right` | Top-right (default) |
| `nf-toast-top-left` | Top-left |
| `nf-toast-bottom-right` | Bottom-right |
| `nf-toast-bottom-left` | Bottom-left |

## Class reference

| Class | Description |
|---|---|
| `nf-toast-region` | Fixed container — stacks toasts vertically |
| `nf-toast` | Individual notification |
| `nf-toast-success` | Primary green left border |
| `nf-toast-error` | Error red left border |
| `nf-toast-warning` | Secondary magenta left border |
| `nf-toast-info` | Tertiary purple left border |
| `nf-toast-dismissing` | Triggers the slide-out exit animation |
| `nf-toast-content` | Flex column wrapper for title + message |
| `nf-toast-title` | Label-font heading |
| `nf-toast-message` | Body text |
| `nf-toast-close` | Dismiss button |
