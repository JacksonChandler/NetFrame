## Class reference

| Class | Description |
|---|---|
| `nf-tag` | Base chip — neutral border and background |
| `nf-tag-primary` | Primary green tint + border |
| `nf-tag-secondary` | Secondary magenta tint + border |
| `nf-tag-tertiary` | Tertiary purple tint + border |
| `nf-tag-close` | `×` button inside the tag — renders via `::before` |
