# Skeleton

Shimmer placeholders for content that is loading. Uses the primary color at low opacity for an on-theme shimmer effect.

## Class reference

| Class | Description |
|---|---|
| `nf-skeleton` | Base — shimmer animation, must be combined with a shape |
| `nf-skeleton-text` | Full-width text line, 0.875rem tall |
| `nf-skeleton-text-short` | Shortens `.nf-skeleton-text` to 60% width |
| `nf-skeleton-heading` | 1.5rem tall heading placeholder |
| `nf-skeleton-avatar` | 2.5rem square avatar placeholder |
| `nf-skeleton-button` | 2.5rem × 8rem button placeholder |
| `nf-skeleton-card` | Full-width, 10rem tall card placeholder |
| `nf-skeleton-image` | Full-width 16:9 image placeholder |
