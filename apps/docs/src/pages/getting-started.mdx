# Getting Started

NetFrame is a pure CSS component library. No JavaScript framework required — drop in the stylesheet and use the classes.

## Installation

```bash
npm install netframe
# or
pnpm add netframe
# or
yarn add netframe
```

## Usage

### Bundler (Vite, Webpack, etc.)

```js
import 'netframe/dist/netframe.css';
```

### CDN / link tag

```html
<link rel="stylesheet" href="https://unpkg.com/netframe/dist/netframe.min.css" />
```

### Sass consumers

If you want access to the raw Sass tokens and mixins in your own `.scss` files:

```scss
// Import everything
@use 'netframe/scss/netframe';

// Or cherry-pick
@use 'netframe/scss/tokens';
@use 'netframe/scss/mixins/neon-glow' as *;
```

## CRT Overlay

The scanline effect is a core identity element. Add `.nf-crt` to `<body>` to apply it globally:

```html
<body class="nf-crt">
  ...
</body>
```

This adds a fixed full-screen pseudo-element with the RGB phosphor + 3px scanline pattern. It uses `pointer-events: none` and `z-index: 9999` so it floats above all content without interfering with interaction.

## Icons

NetFrame uses **Phosphor Thin** icons. Install and reference them:

```html
<!-- CDN -->
<script src="https://unpkg.com/@phosphor-icons/web@2.1.1/src/thin/index.js" type="module"></script>

<!-- Usage -->
<i class="ph-thin ph-terminal"></i>
<i class="ph-thin ph-folder-open"></i>
```

Or install as a package:

```bash
npm install @phosphor-icons/web
```

## Browser support

NetFrame uses `color-mix()`, CSS nesting, and `:focus-visible`. Supported in all evergreen browsers (Chrome 111+, Firefox 113+, Safari 16.4+).
